AMD Ryzen 5 3600
Este procesador tiene seis núcleos y doce hilos, con una velocidad base de 3.6 GHz y una velocidad de turbo de 4.2 GHz. Es un procesador de gama media-alta, adecuado para juegos y aplicaciones que requieren de un alto rendimiento.
AMD A8 9600
Este procesador tiene cuatro núcleos y cuatro hilos, con una velocidad base de 3.1 GHz y una velocidad de turbo de 3.4 GHz. Es un procesador de gama media-baja, adecuado para tareas básicas como navegación web y ofimática.
Intel i3 3230
Este procesador tiene dos núcleos y cuatro hilos, con una velocidad base de 3.2 GHz. Es un procesador de gama media-baja, adecuado para tareas básicas como navegación web y ofimática.
