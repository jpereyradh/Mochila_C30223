
1) ¿Quién inventó el sistema de control de GIT y porqué?

El sistema de control de GIT fue creado por Linus Torvalds, lo creó planteando el uso de poder llevar
registro en los cambios de cada archivo y poder coordinar ese trabajo entre varias personas que estén 
haciendo modificaciones sobre el mismo. Fue pensado en la eficiencia, confiabilidad y compatibilidad 
del mantenimiento de versionen en aplicaciones cuando tienen un gran número de código fuente.
Actualmente GIT es un sistema de control de versiones con funcionalidad completa.
