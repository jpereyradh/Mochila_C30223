CPU

CPU

El procesador (CPU, Central Processing Unit) es el componente más importante dentro del PC. Es el cerebro de todo el funcionamiento del sistema, el encargado de dirigir todas las tareas que lleva a cabo el equipo y de ejecutar el código de los diferentes programas.

BIT

BIT

el bit corresponde a un dígito del sistema de numeración binario y representa la unidad mínima de información. El término es un acrónimo de binary digit.

Archivo/Fichero

Archivo/Fichero

Un archivo es una colección ordenada de datos que tienen entre sí una relación y que se almacenan de forma permanente en un dispositivo de memoria no volátil. Un fichero es un conjunto de información relacionada, grabada en el sistema de almacenamiento secundario y a la que se hace referencia mediante un nombre.


CODIGO FUENTE

CODIGO FUENTE

El código fuente es un archivo o archivos que contienen las instrucciones necesarias, realizadas en programación, que tiene como finalidad compilar posteriormente un programa o programas para que puedan ser utilizados por el usuario de forma directa, con solo ejecutarlo.

TERMINAL

TERMINAL

Es un programa que esta disponible en todos los sistemas operativos, sirve para darle ordenes al sistema, Una terminal es un programa cuyo objetivo principal es leer comandos y ejecutar otros programas

MALWARE

MALWARE

Es cualquier tipo de software malicioso o malintencionado que pueda afectar de alguna manera al usuario o a sus intereses, por ejemplo un spyware que actúe sobre la información del usuario sin previo aviso podría ser considerado un malware.

BYTE

BYTE

Se conoce como byte a la unidad básica de información empleada en la informática y las telecomunicaciones, equivalente a un conjunto ordenado y regular de bits (código binario), generalmente estipulado en 8.


ENCRIPTACIÓN

ENCRIPTACIÓN

La encriptación es una forma de codificar los datos para que solo las partes autorizadas puedan entender la información. En términos técnicos, es el proceso de convertir un texto plano legible para seres humanos en un texto incomprensible, también conocido como texto encriptado.

BIOS

BIOS

También lo podemos llamar sistema básico de entrada y salida, es el primer programa que se inicia al iniciar el sistema, se encuentra alojado en un chip ubicado en la placa madre o motherboard, y se encarga de inicializar y probar todo el harware del sistema informático.

FRAMEWORK

FRAMEWORK

Un framework, o también llamado marco de trabajo, es un conjunto de herramientas usado con frecuencia por los desarrolladores de software. Permiten agilizar procesos ya que nos evita tener que escribir líneas de código repetitivamente.
En la imagen de arriba podemos ver ejemplos significativos, por ejemplo React.

GITHUB

GITHUB

Github es un portal creado para alojar el código de las aplicaciones de cualquier desarrollador, y que fue comprada por Microsoft en junio de 2018. La plataforma está creada para que los desarrolladores suban el código de sus aplicaciones y herramientas, y que como usuario no sólo puedas descargarte la aplicación, sino también entrar a su perfil para leer sobre ella o colaborar con su desarrollo. 

Se trata de una de las principales plataformas para crear proyectos abiertos de herramientas y aplicaciones, y se caracteriza sobre todo por sus funciones colaborativas que ayudan a que todos puedan aportar su granito de arena para mejorar el código.

GIT

GIT

Git es un sistema de control de versiones distribuido, lo que significa que un clon local del proyecto es un repositorio de control de versiones completo. Estos repositorios locales plenamente funcionales permiten trabajar sin conexión o de forma remota con facilidad.

Carpeta / Directorio

Carpeta / Directorio

Una carpeta (también conocida como directorio) es un elemento de informática que puede contener archivos. Puede verse como un gran armario que contiene cajones en los que se guardan archivos.

VIRUS

VIRUS

Un virus informático es un software que tiene por objetivo alterar el funcionamiento normal de cualquier tipo de dispositivo informático, sin el permiso o el conocimiento del usuario principalmente para lograr fines maliciosos sobre el dispositivo. 

ROM

ROM

SSD

SSD

Funciona como unidad de almacenamiento, mucho más rápida que el disco rígido y de menor tamaño. Vienen en dos formatos, el que se conecta por cable sata, y el M.2 que va directo a la mother y es de menor tamaño.

Memoria RAM

Memoria RAM

La memoria RAM es la memoria principal de un dispositivo donde se almacena programas y datos informativos. Las siglas RAM significan "Random Access Memory"

ALMACENAMIENTO EN LA NUBE

ALMACENAMIENTO EN LA NUBE

El almacenamiento en la nube es un modelo de computación en la nube que permite almacenar datos y archivos en Internet a través de un proveedor de computación en la nube, al cual se accede mediante la red pública de Internet o una conexión de red privada dedicada.

LAN

LAN

LAN son las siglas de Local Area Network, Red de área local. Una LAN es una red que conecta los ordenadores en un área relativamente pequeña y predeterminada (como una habitación, un edificio, o un conjunto de edificios).

HACKER

HACKER

El concepto tiene dos grandes acepciones ya que puede referirse a un pirata informático (una persona que accede de manera ilegal a un sistema para tomar su control u obtener datos privados) o a un experto que se encarga de proteger y mejorar la seguridad informática.




FIREWALL

FIREWALL

Un firewall es un sistema diseñado para proteger las redes privadas del acceso no autorizado y no verificado en una conexión a Internet.

GABINETE

GABINETE

Funciona como caja para todo el hardware de la computadora, vienen de diferentes tamaños, mini ITX, micro ATX, ATX y full tower, este último destinado a ensambles mas complejos como alguno que posea refrigeración líquida con bomba.

DIRECCION IP

DIRECCION IP

El término "IP" se refiere a "Protocolo de Internet" (Internet Protocol, en inglés), que es un conjunto de reglas que gobiernan la comunicación de datos a través de redes de computadoras.
El Protocolo de Internet es responsable de asignar direcciones únicas a cada dispositivo conectado a Internet y de enrutar los datos a través de la red para que lleguen a su destino correcto. Estas direcciones únicas son conocidas como direcciones IP.
Las direcciones IP se utilizan para identificar y comunicarse con dispositivos en redes como Internet. Hay dos versiones principales de direcciones IP: la versión 4 (IPv4) y la versión 6 (IPv6). IPv4 es el sistema de direcciones IP original y todavía se usa mucho, pero IPv6 fue desarrollado para abordar la escasez de direcciones IP en el futuro.
En resumen, el IP es un protocolo que permite la comunicación de datos a través de redes de computadoras y las direcciones IP son identificadores únicos utilizados para identificar y comunicarse con dispositivos en estas redes.

CODIGO MAQUINA

CODIGO MAQUINA

El código de máquina es un lenguaje de programación de computadora que comprende instrucciones hexadecimales o binarias a las que las computadoras pueden responder directamente. El código de máquina está escrito en un lenguaje de máquina. Por lo tanto, una máquina, es decir, una computadora, puede ejecutarlo sin ningún tipo de traducción o conversión.

También nos referimos a él como lenguaje de máquina. A las instrucciones que existen en código de máquina las llamamos instrucciones de máquina.
